# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a **GitHub issue** in this repository.

When reporting a vulnerability, please include:

- A clear and concise description of the issue  
- Steps to reproduce (if applicable)  
- Any relevant logs, screenshots, or proof-of-concept code  

We appreciate all contributions that help improve the security and reliability of this project.

Thank you for helping keep this project secure!
